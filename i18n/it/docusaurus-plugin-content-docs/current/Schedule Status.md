I Jobs hanno molti stati diversi

* Alcuni tipici stati di Jobs includono:
    * Qualifying
    * Wait to Start
    * Time
    * Dependency
    * Thresholds/Resources
    * Machine
    * Running
    * Finished OK
    * Marked Finished OK
    * Failed
    * Marked Failed
    * On Hold
    * Cancelled
    * Skipped (or Job to be Skipped)
    * Under Review
    * Fixed
